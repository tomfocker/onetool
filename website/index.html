Crystal Concept

它不是把工具堆在一起，而是把系统能力压成一块透明内核。

OneTool 服务的是那些整天在桌面里切任务、切窗口、切系统状态的人。真正重要的不是功能条目有多少，而是这些动作能不能被压缩成一条更短的工作路径。

Control 一块前台可见的桌面控制晶体

代理切换、WSL 状态、截图录制、批量处理与悬浮收纳，被重新放进同一块冷静表面里。

Material 冷白 / 冰蓝 / 微偏光

“越靠近系统深水区，界面越应该透明、克制、能被一眼拆解。”

System Layer

深系统能力，不该继续藏在命令行和多层设置面板后面。

Proxy Surface 本地代理管理

读取系统状态、切换 HTTP / HTTPS 或 SOCKS5、本地端口预设、旁路规则、快速打开系统代理页。

WSL Surface WSL 虚拟机管理

查看发行版、默认项、运行状态，直接启动、终止、切换实例。

Daily Flow 媒体捕获 + 高频效率

叠加截图、屏幕录制、屏幕取色、批量重命名、剪贴板管理和悬浮球一起组成真实桌面工作流。
