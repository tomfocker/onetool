# OneTool v0.3.1

这次更新重点完善桌面日历小组件，并继续打磨空间分析的 NTFS 极速扫描体验。

## 主要更新

- 新增桌面日历小组件：支持独立窗口、开机后恢复显示、本地日程提醒和工具箱内日历数据同步。
- 优化日历小组件显示：增加透明度、背景模糊、边框强度等显示选项，默认玻璃效果更柔和。
- 优化空间分析：NTFS 极速扫描增加更清晰的进度提示、动态目录预览和完成后的三级目录结果。
- 提升空间分析稳定性：加强提权扫描资源校验，修复中文路径输出和扫描事件读取相关问题。
- 优化扫描性能：限制 NTFS 扫描过程中的预览树大小和刷新频率，避免动态展示拖慢快扫。

## 修复与优化

- 修复桌面日历小组件背景中出现多余矩形磨砂层的问题。
- 修复日历小组件显示参数不能持久化或跨窗口同步的问题。
- 修复 NTFS 极速扫描过程早期缺少明确状态反馈的问题。
- 修复空间分析中目录 / 文件统计可能不准确的问题。
- 修复 NTFS 扫描器资源缺失时错误提示不够明确的问题。

## 升级提醒

- 当前 Windows 构建尚未代码签名，首次运行可能出现 SmartScreen 或“未知发布者”提示。
- 安装版支持应用内更新；便携版不支持应用内自动更新。
- 桌面日历小组件和日历应用当前都使用本地数据，暂不支持云同步。

## 验证状态

- 本地测试：534 项 JS 测试通过；NTFS 原生测试 19 项单元测试 + 1 项集成测试通过。
- 本地生产构建：通过。
- Windows 安装包 / 便携版构建：本机打包阶段受 Windows 符号链接权限限制影响，需在具备符号链接权限的发布环境完成。
