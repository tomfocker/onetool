# v0.2.0 升级要点

`v0.2.0` 是一次明显偏向“基础能力升级 + Windows 体验收口”的版本，不只是功能堆叠。

## 重点升级

- 引入全局 `OpenAI-compatible` LLM 配置，多个工具共用同一套 `Base URL / API Key / Model`
- 截图工具升级为两种模式：
  - `仅提取文字`
  - `OCR + 翻译`
- 首批 AI 增强能力落地：
  - 截图 OCR / 翻译
  - 批量重命名建议
  - 系统诊断建议
  - 空间清理建议
- 主进程启动链完成一轮模块化拆分，Windows 运行时结构更清晰
- `settings` / `store` 加入 schema-driven migration 基线，后续版本升级更稳
- 辅助窗口运行时完成一轮轻量化，截图、取色、录屏选区、截图选区与主应用入口解耦
- Windows 打包链路补齐 FFmpeg 运行时准备，录屏、B 站下载、系统诊断统一走稳定路径
- 空间分析加入按需管理员提权的 NTFS 极速扫描链路，普通权限主应用不再被整体验证拖成管理员

## 用户可感知变化

- Windows 安装版的更新链路继续保留，便携版仍不启用自动更新
- 截图翻译交互重新收口，优先保证“可框选、可识别、可关闭、结果不串场”
- 空间分析在选择 NTFS 盘根时，会在需要时请求管理员权限，以执行真正的 NTFS 极速扫描

## 发布说明建议

- 当前仍为未签名 Windows 构建，首次运行可能出现 `SmartScreen` 或“未知发布者”提示
- 软件内自动更新仅适用于 Windows 安装版；便携版不支持应用内升级
- 已安装的 `v0.1.0` 只有在本次正式发布的版本号高于 `0.1.0` 时，才会在应用内识别到新版本
